**部队掷骰系统说明:**
  计算公式: "基础成功率 + (士气值-50)/2.5 + (玩家等级-10)/2 + 玩家军衔*5 + (当前地形适应性-5)*4 + (补给状态-50)/3.3 + (体力值-50)/5 + (训练等级-5)*2"
  计算示例: "基础50 + 士气8 + 等级0 + 军衔5 + 地形0 + 补给9 + 体力8 + 训练0 = 80，掷骰≤80成功"
  影响因素:
    - "士气值: 每2.5点士气影响1点成功率"
    - "玩家等级: 每2级影响1点成功率"
    - "玩家军衔: 每级影响5点成功率"
    - "地形适应性: 每点影响4点成功率"
    - "补给状态: 每3.3点影响1点成功率"
    - "体力值: 每5点影响1点成功率"
    - "训练等级: 每级影响2点成功率"

**军衔映射关系:**
  - 0: 士兵
  - 1: 队长
  - 2: 骑士团长
  - 3: 军团长

**部队信息:**
  * 部队应视为一个特殊角色生成
  * 基础数值HP，MP，SP参考部队默认阶层单人数值*部队人数
  * 部队默认阶层与装备:
    - 新兵: 第一阶层，普通品质装备
    - 老兵: 第二阶层，普通品质装备和部分稀有品质装备
    - 精锐：第二阶层与少量第三阶层，稀有品质装备
    - 禁卫: 第三阶层，稀有品质装备
  * 部队五维按照层级合理给出，单人数值*(部队人数/50)
  * 部队默认装备按照层级和部队属性默认给出，数值合理，无特殊技能(法师部队可以有法术)
  * 合理考虑其他信息如士气，补给，地形适应度等
  * 特殊部队特性:
    - **弓箭手部队:**
      - 行动顺序:+5修正
      - 命中:+10修正
      - 对轻甲单位伤害+50%
      - 需要弹药补给
      - 近战防御较弱

    - **法师部队:**
      - 行动顺序:-3修正
      - 命中:+5修正
      - 对重甲单位伤害+50%
      - 消耗法力值
      - 崩溃时影响全军士气-20%

    - **战争巨兽:**
      - 行动顺序:-10修正
      - 命中:-5修正
      - 对建筑伤害+100%
      - 高生命值，影响全军士气
      - 崩溃时影响全军士气-30%

    - **火炮部队:**
      - 行动顺序:-15修正
      - 命中:-10修正
      - 对集群单位伤害+80%
      - 需要弹药补给
      - 移动缓慢，部署需要时间

  * 部队品质升级系统:
    - **升级阈值:**
      - 新兵 → 老兵: 10000经验值
      - 老兵 → 精锐: 30000经验值
      - 精锐 → 禁卫: 60000经验值,且需要特殊机遇
    - **升级效果:**
      - 品质升级时训练等级+1
      - 基础属性永久提升
      - 解锁新的战术能力
    - **品质等级效果:**
      - 新兵: 基础属性，无特殊加成
      - 老兵: 士气+10%，经验获取+10%
      - 精锐: 士气+20%，经验获取+20%，解锁特殊战术(如侧翼袭击，埋伏等)
      - 禁卫: 士气+30%，经验获取+30%，解锁高级战术(特殊的阵型，独特的技能等)，免疫士气崩溃

  * 招募新兵惩罚系统:
    - **招募影响:** 每次大规模招募新兵，累计经验值减少20%
    - **品质降级:** 当累计经验值低于当前品质等级阈值时，品质自动降级
    - **恢复难度:** 降级后需要重新积累经验值才能恢复

**能独自对左右战局的强者:**

    - 第六层级: 任何情况都应单独列出
