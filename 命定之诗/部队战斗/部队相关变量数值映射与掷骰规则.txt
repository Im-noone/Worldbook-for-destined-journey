# 新增掷骰系统说明
部队掷骰系统说明:
  基础规则: "使用d100系统，计算总修正值后与掷骰结果比较，≤修正值即为成功"
  计算公式: "基础成功率 + (士气值-50)/2.5 + (玩家等级-10)/2 + 玩家军衔*5 + (当前地形适应性-5)*4 + (补给状态-50)/3.3 + (体力值-50)/5 + (训练等级-5)*2"
  计算示例: "基础50 + 士气8 + 等级0 + 军衔5 + 地形0 + 补给9 + 体力8 + 训练0 = 80，掷骰≤80成功"
  影响因素:
    - "士气值: 每2.5点士气影响1点成功率"
    - "玩家等级: 每2级影响1点成功率" 
    - "玩家军衔: 每级影响5点成功率"
    - "地形适应性: 每点影响4点成功率"
    - "补给状态: 每3.3点影响1点成功率"
    - "体力值: 每5点影响1点成功率"
    - "训练等级: 每级影响2点成功率"

军衔映射关系: 
  - "0": string. 士兵 soldier
  - "1": string. 队长 sergeant
  - "2": string. 骑士团长 officer
  - "3": string. 军团长 general
地形映射关系:
  - "0": string. 平原
  - "1": string. 森林
  - "2": string. 山地
  - "3": string. 攻城
  - "4": string. 守城
阶段映射关系:
  - "0": string. 进攻
  - "1": string. 防守
  - "2": string. 相持
  - "3": string. 撤退
